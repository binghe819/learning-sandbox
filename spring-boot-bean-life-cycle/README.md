## 주제
빈 생명주기란 빈이 인스턴스화되는 시기와 방법, 생성되고 소멸되기 까지의 수행하는 작업, 소멸되는 시기 및 방법을 의미한다.

이번 학습 테스트의 목적은 빈 생명주기와 콜백 메서드를 직접 알아보기 위함이다.
